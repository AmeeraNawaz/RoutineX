
# RoutineX

A smart productivity and routine management platform built using ASP.NET Core MVC.

## Features

### Task Management
- Create tasks
- Edit tasks
- Delete tasks
- Task categories
- Priority levels
- Due date and time tracking

### Productivity Dashboard
- Total Tasks
- Completed Tasks
- Pending Tasks
- Overdue Tasks

### Analytics
- Weekly Productivity Analysis
- Monthly Productivity Analysis
- Yearly Productivity Analysis
- Dynamic Chart.js graphs

### Smart Features
- Automatic overdue detection
- Progress tracking
- Productivity insights

### Upcoming Features
- Voice Assistant Task Creation
- AI Task Scheduling
- Smart Recommendations
- Achievement System
- Productivity Score
- Focus Mode
